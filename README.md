# Qt sample gallery application

Contains test gallery application to view image files in the specified folder.

Supports the next operations:

* open specified folder with images
* jump to the next image in folder
* jump to the previous image in folder
* change brightness for the current image

## Required libraries

1. CMake 3.19+
2. Qt 6.5
3. OpenCV

## Building

Build project

> $ cmake build -B build .

> $ cd build

> $ make
